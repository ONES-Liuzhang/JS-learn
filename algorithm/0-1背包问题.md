经典的动态规划问题
题干： 有一个重量为 W 的背包，和 N 个物品，已知第 i 个物品的重量为 wt[i]，价值为 val[i]。请计算在不超重的情况下最多可以装入物品的总价值为多少

## 明确[状态]和[选择]

1. 状态
   重量 wt
   价值 val

2. 选择
   装进背包
   不装进背包

## 明确 dp 数组的含义

val = dp[i][w]

dp[i][w]表示对前 i 个物品，背包容量为 w 时，可以装下的最大价值

## 状态转移方程

当背包容量为 w 时

1. 对第 i 个物品，如果不装进背包
   dp[i][w] = dp[i-1][w]

2. 对第 i 个物品，如果装进背包
   dp[i][w] = dp[i-1]w-wt[i-1]] + val[i-1]

最优价值 = max(dp[i-1][w], dp[i-1]w-wt[i-1]] + val[i-1])

## base

dp[0][w] = 0
dp[i][0] = 0
