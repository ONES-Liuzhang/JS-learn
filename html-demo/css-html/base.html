Inline Block:

描述：可以设置宽高、padding、margin

inline:

行内元素分替换元素和非替换元素

  • 替换元素:像img、input这种用作其他内容占位符的元素
  • 非替换元素:内容包含在文档内的元素，像p、div等
非替换元素：

无法设置宽高，宽高由元素内容决定

行内非替换元素可以设置margin-left和margin-righ，上下外边距不生效

内边距生效，但是不会撑开父元素，设置背景时可见

span
替换元素：

img是行内替换元素，可设置宽高

内外边距都会影响宽高
